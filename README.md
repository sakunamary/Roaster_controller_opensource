全热风咖啡烘豆机主控板，适合自己diy用。
2路温度采集：bt和inlet
温度采集使用max6675 k型热电偶
无线连接artisan
0-10v输出控制可控硅
风机使用12/24v无刷控制。
留有I2C接口可以接扩展传感器、oled等。
目前开源板程序功能如下：
1为无线接artisan记录温度数据
2手动控制风力 火力。
